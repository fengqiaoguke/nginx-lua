### 添加标签接口

```
POST /tag
```

### 参数

| **参数** | **类型** | **描述** |
| :--- | :--- | :--- |
| name | string | 标签名称 |
| uid | int | 用户id |
| fid | int | \[可选\]上级标签id |

### 返回参数

| **参数** | **类型** | **描述** |
| :--- | :--- | :--- |
| - |  |  |

### 返回格式

```js
HTTP/1.1 200 OK
{
"code": "0",
"msg": "提示信息",
"data":{}
}
```

### 例子

```js
http://www.domain.com/v1/tag
```
