### 编辑笔记接口

```
PUT /note/:id
```

### 参数

| **参数** | **类型** | **描述** |
| :--- | :---: | :--- |
| title| string | 标题 |
| content | text | 笔记内容 |
| tag_id | int |  标签id |



### 返回参数

| **参数** | **类型** | **描述** |
| :--- | :--- | :--- |
| - |  |  |

### 返回格式

```js
HTTP/1.1 200 OK
{
"code": "0",
"msg": "提示信息",
"data":{}
}
```

### 例子

```js
http://www.domain.com/v1/note/1
```
